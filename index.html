Interfaces

IAuth Interface for authentication mechanisms.
IPermissionChecks Interface for permission checks.
IRestSerializeFormatter The interface for serialize formatters for the rest module provides defines a method for working with the given data.
IRestSerializer Provides methods for serializing data.
IRestValidator Interface for data validators in the rest api.

Classes

ApiSession
AuthFactory Factory for different kind of rest authenticators.
BaseRestController Base class for the rest resource controllers.
DataListExtension Useful extensions for data lists.
HtmlSerializer Serializer for html.
HttpAuth Authentication mechanism using a BasicAuth request.
JsonSerializer Serializer for json.
JwtAuth Authentication mechanism using a token in the request header and validates it on every request.
NestedResourceRestController The nested resource controller can help you to avoid some boilerplate in nested resources in you rest api.
PaginationExtension Class PaginationExtension
RestSystemException The system exception can be used for showing a system error like a missing file or a broken connection.
RestTest Rest test class can work as base class for your functional tests. It provides some helpful methods to test your rest api more easily.
RestUserException The user exception can be used for indicating the wrong usage of the rest api.
RestValidatorHelper Some generic validators for incoming data.
SerializerFactory Factory for different kind of rest serializers.
SessionAuth Authentication mechanism which uses the internal Silverstripe authentication (session based).
SessionController ApiSession controller is the controller for the session resource.
SessionFormatter The session formatter can format a session into a representation which is consumable from a IRestSerializer.
SessionValidator Class SessionValidator
SilverstripePermission Implements the IPermission interface and uses the Silverstripe permission system.
SlugableExtension Extension for data objects which should be identifiable by a slug.
TestHelper Test Helper class provides some helpful functions for tests.
TokenAuth Authentication mechanism using a token in the request header. Valid tokens are saved in cache.
XmlSerializer Serializer for xml.
YamlSerializer Serializer for yaml.
